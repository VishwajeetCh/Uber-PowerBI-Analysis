# 🚖 Uber Data Analysis Dashboard (Power BI)

## 📌 Project Overview
This project analyzes Uber ride data using Power BI to uncover insights into ride trends, revenue patterns, and customer behavior.

The dashboard provides interactive visualizations to help understand business performance and demand patterns.

---

## 🎯 Objectives

- Analyze total rides and revenue trends
- Identify peak booking hours and demand patterns
- Evaluate trip distance and fare distribution
- Understand customer usage behavior
- Provide actionable business insights
